Desenvolvimento Web Dezembro 2024

Descrição

Este script Python utiliza o nmap para executar scans de rede direcionados, com foco no TCP e scans limitados de portas UDP para identificar e alertar sobre possíveis vulnerabilidades nos serviços de rede. Foi concebido para avaliar rapidamente a segurança de serviços comuns como HTTP, FTP e SMTP, analisando as versões de serviço em relação a vulnerabilidades conhecidas.
